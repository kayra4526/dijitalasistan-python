# 🚀 Kişisel Dijital Asistan (GUI)

Bu proje, Python ve **CustomTkinter** kullanılarak geliştirilmiş, modern ve karanlık temalı (dark mode) bir masaüstü kişisel asistan uygulamasıdır. Kullanıcıların tek bir pencere üzerinden hem kişisel bütçelerini hem de günlük görevlerini/notlarını yönetmelerini sağlar.

## ✨ Özellikler
- **Modern Arayüz:** CustomTkinter ile tasarlanmış şık, sekmeli (tabview) yapı.
- **💰 Bütçe Takibi (Sekme 1):** - Gelir güncelleme, dinamik kategori bazlı harcama ekleme.
  - Anlık bütçe analizi ve eksi bakiye uyarı sistemi.
- **📓 Ajanda & Görevler (Sekme 2):**
  - Tarih bazlı günlük notları tutma.
  - Görev ekleme, tamamlama (`✅`) ve silme (`🗑️`).
- **Gizlilik Odaklı Veri Kalıcılığı:** Tüm veriler yerel olarak `.json` formatında saklanır. 

## 🛠️ Kurulum ve Çalıştırma

Gerekli kütüphaneyi kurun:
pip install -r requirements.txt

Uygulamayı çalıştırın:
python kisisel_asistan.py
